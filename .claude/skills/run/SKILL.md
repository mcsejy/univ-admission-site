이 프로젝트는 빌드 단계가 없는 순수 정적 파일 앱이다.

## 실행 방법

**방법 1 — VS Code Live Server (권장)**
1. VS Code에서 `index.html` 파일을 열기
2. 우클릭 → "Open with Live Server" 선택
3. 브라우저에서 자동으로 열림 (기본: `http://127.0.0.1:5500`)

**방법 2 — Python HTTP 서버**
터미널에서 프로젝트 루트 디렉토리로 이동 후:
```
python -m http.server 8000
```
브라우저에서 `http://localhost:8000` 접속.

**방법 3 — 파일 직접 열기**
`index.html`을 브라우저로 드래그하거나 더블클릭. LocalStorage는 정상 동작하지만 일부 브라우저에서 `file://` 프로토콜 제한이 있을 수 있다.

## 확인 사항

앱이 열리면:
- 통계 카드 (전체/완료/진행중/달성률) 표시 확인
- 항목 추가/체크/수정/삭제 동작 확인
- LocalStorage에 데이터가 저장되는지 확인 (새로고침 후 데이터 유지)
